synthesisapi.routers.update_paragraphs()[source]¶

Updates the list of paragraphs and queries (POST/PUT ROUTE)

(Params are all REST params)

Parameters:
  • update_type (int) – 1 = addition, -1 subtraction
  • material_id (unicode) – Relevant material ID
  • paragraph_text (unicode) – raw text of the paragraph
  • rank (int) – Relevance ranking for this paragraph (i.e. relevance to target material)
  • feature_vector (unicode) – Each element in array concatenated by ”,” e.g. “1,2,3” = [1,2,3]
  • is_recipe (int) – 0 = not a recipe paragraph, 1 = is a recipe paragraph
  • doi (unicode) – DOI of the paper the paragraph belongs to
Returns:

Success or error (returns error message on error)
Return type:

json
